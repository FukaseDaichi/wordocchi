# ワードッチ ルール定義書

最終調査日: 2026-05-25  
対象: アークライト『ワードッチ』を補助する非公式デジタルツール

## 位置づけ

この文書は、アークライト『ワードッチ』の公開情報をもとに、Web ツールへ落とし込むためのルールを整理したものです。公式説明書そのものではありません。カード本文、収録ワード、収録キジュンの網羅的な転記は行わず、実装では「ユーザーが入力するデータ」または「独自に作成したサンプルデータ」を扱います。

## 商品・プレイ条件

- プレイ人数: 3-8人
- プレイ時間: 5-10分
- 対象年齢: 6-99歳
- 内容物: カード81枚、説明書1枚
- カード構成: 公開情報では、カード表面にワード、裏面にキジュンがある。PR TIMES ではワード243個、キジュン81問の収録と説明されている。
- ゲームの性質: 親だけが知る「ヒミツのキジュン」に対して、子がさまざまなワードを出し、親の主観判定から傾向を探る会話ゲーム。

## 用語

- 親: ラウンドの判定者。ヒミツのキジュンを知っており、ワード同士を比較する。
- 子: 親以外のプレイヤー。ワードを出し、親の判定を観察する。
- ヒミツのキジュン: 親が選び、最後まで子に公開しない比較基準。
- 暫定チャンピオン: 調査フェイズ中に最もキジュンへ合っていると親が判定した現時点の勝ち残りワード。
- 優勝ワード: 決選フェイズで親が最もよいと選んだワード。

## ラウンド構造

1. セットアップ
2. 調査フェイズ
3. 決選フェイズ
4. キジュン公開・感想戦

## セットアップ

1. 親を1人決める。
2. 親以外を子とする。
3. 親にカード2枚配る。
4. 親は自分のカード2枚のキジュン面をこっそり確認し、そのうち1つを今回のヒミツのキジュンとして選ぶ。
5. ヒミツのキジュンは、ラウンド終了まで子に見せない。

## 調査フェイズ

1. 親は自分のカードのワード面から、最初の暫定チャンピオンにするワードを1つ選んで読み上げる。
2. 子は、思いついたワードを順番に親に提示する。
3. 親は、提示されたワードと現在の暫定チャンピオンをヒミツのキジュンに照らして比較し、どちらが勝つかを主観で決める。
4. 提示されたワードが勝った場合、そのワードが新しい暫定チャンピオンになる。負けた場合、暫定チャンピオンは変わらない。
5. 2-4をテンポよく繰り返す。公式説明では順番や回数制限は設けられていない。
6. 制限時間がきたら、調査フェイズを終了する。

## 決選フェイズ

1. 子は、調査フェイズで出ていないワードから、親が選びそうなワードを1つ考える。
2. 全ての子が準備できたら、親の合図で同時に発表する。
3. 親は発表されたワードを1つずつ確認し、ヒミツのキジュンに最も合うと思う優勝ワードを選ぶ。
4. 優勝ワードを出した子がそのラウンドの勝者になる。
5. 親はヒミツのキジュンを公開し、全員で感想を話す。

## 判定原則

- 親の主観判定を正とする。
- 子がキジュンを完全に当てる必要はない。判定履歴から「なんとなく傾向をつかむ」ことが遊びの中心になる。
- 正解・不正解を厳密に裁定するゲームではなく、発想や会話の盛り上がりを優先する。
- キジュンはラウンド終了まで秘匿する。
- 決選フェイズでは、過去に出たワードの再利用を避ける。
